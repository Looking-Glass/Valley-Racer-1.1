HOW TO UPDATE HYPERCUBE:


Here I will list steps to take to update the hypercube plugin in valley racer
DEFAULT SETTING CHANGES:
* Slicing: Hard
* Overlap: 0
* Black Point: set to black
* Perspective Factor: 0.4 (0.0 on hi-score scene)
* Tube Factor: 0.7 (0.0 on hi-score)
* Hypercube_rtt camera culling: everything
* HYPERCUBE mesh renderer: disable


ADDING PERSPECTIVE FACTOR:
Current perspective factor: 0.4
Current tube factor: 0.7


STEP 1: add some fields

       public float perspectiveFactor = 0.4f;
       public float tubeFactor = 0.7f;
       float[] perspectiveCameraSizes;

for (int i = 0; i < localCastMesh.slices; i++)

STEP 4: also in resetSettings(), replace the guts in the for loop with this:


var newSliceDepth = i * sliceDepth * (i * perspectiveFactor + 1) - overlap;
